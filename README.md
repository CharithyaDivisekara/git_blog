This project is a simple Git Basics tutorial website created to make learning Git easy and clear.
It explains Git concepts step by step using real commands, examples, and simple explanations.

The website is built as a single-page, fully responsive site and is suitable for anyone who wants to understand how Git works in real projects.

Project Features

    Easy explanations of Git concepts
    Real Git command examples with terminal-style output
    Covers basic to practical topics (workflow, branches, undo actions, collaboration)
    Smooth navigation between sections
    Mobile-friendly and responsive design
    
Technologies Used
  HTML
  CSS (Tailwind CSS)
  JavaScript

How to Use This Project
  1. Clone the repository: git clone https://github.com/CharithyaDivisekara/git_blog.git
  2. Open the project folder
  3. Open the index.html file in any modern browser
  4. Start learning Git
